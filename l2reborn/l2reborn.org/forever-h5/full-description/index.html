Server info
Server time is: UTC +0
The server will be based in seasons with an eternal / never wipe server. and new seasons that will be merged on it over the time
It will have 2 seasons per year that will be merged into the main server at the end of each season
Autum Season
Starts 18th October until merge with main server
Spring Season
Starts 11th April until merge with main server

The online time of the server is guaranteed regardless of the amount of players, 6 months per season and an eternal server that will be never wiped

The main difference between Reborn Forever H5 x15 and other midrate servers is that you cannot buy ingame advantage such as exp bonuses, items, enchants, adenas… everything is earn by self effort and dedication, no more P2W. We will advertise the server in topsites, discord, mail, google ads and facebook ads but we won’t never pay any clan or cp to play our server or give any special advantage in the game for “important clans”, “popular players” or similars.

Additional Features and changes
Gameplay
  • Autolearn skills (Divine Inspiration and forgotten scroll skills are excluded).
  • Auto loot.
  • Drop down on monsters to see the droplist.
  • Offline shop (duration 5 days before being kicked).
  • Auto Augment with wished augment selection.
  • Auto item enchant.
  • Auto attribute enchant.
  • Auto potions.
  • Inventory slots 100.
  • Summons get buffs from master (god like).
  • Sub class up to level 85.
  • Certification skills on sub classes.
  • All classes have 1 additional sell slot.
Skills
